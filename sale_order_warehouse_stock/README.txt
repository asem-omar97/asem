# Sale Order Warehouse Stock

This module enhances sale order lines by displaying real-time stock availability across multiple warehouses.

## Features

- **Multi-warehouse Support**: Shows available stock from all warehouses
- **Color-coded Indicators**: 
  - Green: Sufficient stock
  - Yellow: Low stock (less than 2x ordered quantity)
  - Red: Insufficient stock
- **Real-time Updates**: Stock quantities update automatically
- **Detailed View**: Shows free stock, reserved, and total quantities
- **Product View**: Added warehouse stock summary in product form
- **Multi-language Support**: Arabic/English interface

## Installation

1. Copy the module to your Odoo addons directory
2. Update the apps list
3. Install "Sale Order Warehouse Stock"

## Usage

1. Create or edit a sale order
2. Add products to order lines
3. View available stock per warehouse in the "Available Stock" column
4. Hover or click for detailed information

## Technical Requirements

- Odoo 18
- Sales module installed
- Inventory module installed
